# ProofPath Community Call

We are opening ProofPath for community experiments.

ProofPath is a causal gateway for AI-agent actions.

> HTTPS protects the connection. ProofPath protects the meaning of the action.

## What we built

ProofPath now has:

- protocol draft;
- Rust verifier;
- Axum gateway;
- upstream forwarding;
- hash-chained JSONL audit logs;
- dangerous AI-agent action demo;
- real-model-agent demo;
- community experiment levels.

## What we need

We need people to test, break, critique, and extend it.

Pick a level:

```text
L0 — Reader feedback
L1 — Local gateway demo
L2 — Real model agent demo
L3 — Red-team the model/action boundary
L4 — Build an integration
L5 — Protocol and threat model critique
```

Start here:

- Repo: https://github.com/safal207/ProofPath

## Useful feedback

We want to know:

- where setup breaks;
- where the protocol is confusing;
- where the boundary is too weak;
- which threat cases are missing;
- what would make this useful in real systems;
- what should be removed or simplified.
